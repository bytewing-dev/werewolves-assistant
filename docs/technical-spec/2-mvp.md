# 2. MVP (Minimum Viable Product) - Simplified Version

## 2.1 Core Features

### Basic Game Setup
1. **Simple Composition Management**
   - Pre-defined village compositions
   - Basic role selection
   - Support for common player counts (6-12 players)

2. **Role Distribution**
   - Simple player name entry
   - Basic card selection system
   - Clear role display with basic rules

3. **Game Flow**
   - Simple day/night cycle
   - Basic timer for phases
   - Player status tracking (alive/dead)

### Game Master Tools
1. **Game Overview**
   - List of players and their status
   - Current phase display
   - Basic game controls (next phase, end game)

2. **Night Phase**
   - Simple role action sequence
   - Basic action resolution
   - Player elimination tracking
