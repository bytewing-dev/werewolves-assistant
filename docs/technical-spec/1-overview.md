# 1. Overview

## 1.1 Project Objective

A web application to assist in running games of "Werewolves of Millers Hollow". The app will focus on:
- Basic game management
- Simple role distribution
- Phase tracking
- Player status monitoring

## 1.2 Target Users

### Game Master
- Needs help managing game phases
- Wants to track player status
- Requires simple role distribution tools

### Players
- Need to see their role clearly
- Want to check role description
- Must understand basic game rules

## 1.3 Usage Mode

### Single Device Approach
- One device (tablet or large phone) for the game
- Passed between game master and players
- Simple interface for all users

### Basic Flow
1. **Game Setup**
   - Create new game
   - Enter player names
   - Distribute roles

2. **Game Play**
   - Track day/night phases
   - Manage player status
   - Follow game events

## 1.4 Core Principles

### Simplicity
- Clear interface
- Basic functionality
- Easy navigation
- Essential features only

### Reliability
- Stable core features
- Basic error handling
- Simple data management
- Consistent experience
